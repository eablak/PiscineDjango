Django Template Engine

Django uses a template engine to combine Python data with HTML pages. Templates help create dynamic web pages easily.

1. Blocks

Blocks are used to insert content into a base layout. They are defined in base.html and filled in other templates.

{% block content }
    <p>This is my page content.</p>
{% endblock }

2. for ... in Loops

You can loop over a list of items passed from the view.

{% for fruit in fruits }
    <p>{{ fruit }}</p>
{% endfor }

This will show each fruit in the list.

3. if Control Structures

You can show different content depending on conditions.

{% if user.is_authenticated }
    <p>Welcome, {{ user.username }}!</p>
{% else }
    <p>Please log in.</p>
{% endif }

4. Displaying Context Variables

Variables from the view can be shown using double curly braces.

{{ name }}

For example, if name = "Alice", it will display: Hello, Alice!

Summary

  • Blocks let you insert content into base templates
  • for loops repeat over a list
  • if shows content based on conditions
  • {{ variable }} displays data passed from Python
